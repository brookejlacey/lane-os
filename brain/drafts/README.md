# brain/drafts

The one path EVERY lane is allowed to write. This is the shared inbox.

- A code lane or desk that needs to record a cross-cutting fact stages it here.
- `ideas/` holds raw captures from the `/spark` skill.
- A spine session later merges drafts into the right brain files (via a `/catchup`
  skill or by hand), then clears them.

Nothing here is canonical. It is staging, not state.
