# /spark

Capture only. Does NOT build, plan, or switch the current work.

1. Write a timestamped file to `brain/drafts/ideas/` with: the raw dump, a one-line
   restatement, the originating lane, and a maturity tag (raw / sketched / ready).
2. Return a single confirmation line and get back to what you were doing.

`brain/drafts/ideas/` is the one path every lane may write, so `/spark` works from a
code lane, a desk, or the spine. Maturing an idea into a real lane is the separate
`/spin-lane` skill.
