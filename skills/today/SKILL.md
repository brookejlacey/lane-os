# /today

1. Read `brain/ACTIVE_NOW.md`, `brain/CONCERNS.md`, and recent `brain/WEEKLY_LOG.md`.
2. Synthesize a focused **3-bullet** "what to work on today", weighted by category
   (your highest-leverage category first), not by what is freshest in memory.
3. Ask ONE anchoring question (what landed since last session) if useful, then stop.

Keep it short. The point is a clear start, not a status report.
